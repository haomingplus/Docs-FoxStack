## 部署你的 Astro 站点
**[点击查看Astro官方部署指南](https://docs.astro.build/zh-cn/guides/deploy/)**

**将博客部署至 Vercel, Netlify, GitHub Pages, Cloudflare Pages, EdgeOne Pages 等。**

## 指令

框架预设：	`Astro`

根目录：	`./`

输出目录：	`dist`

构建命令：	`pnpm run build`

安装命令：	`pnpm install`
